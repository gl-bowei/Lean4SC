This smart contract is a **token contract** designed for minting and managing a collection of NFTs. The main functions include **minting NFTs, reserving NFTs for the owner, updating base URI, setting royalties, and controlling the minting status**. The main states of the contract are **Minting Active and Minting Inactive**. The logical process involves **activating or deactivating minting, checking the current supply against the maximum supply, and allowing users to mint NFTs, up to a specified limit, while paying a set price**. The main variables include **MAX_SUPPLY, PRICE, MAX_PER_MINT, baseTokenURI, isActive, royalties, and beneficiary**. Key functions include **mintNFTs, reserveNFTs, setBaseURI, setRoyalties**, and **withdraw**. Main events include **minting actions** and **royalty calculations**.

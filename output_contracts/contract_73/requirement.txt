This smart contract is an [NFT contract]. Its main functions include [minting public and private NFTs, setting the base URI for NFTs, and retrieving metadata]. The key states of the contract are [Public Minting, Private Minting, and Metadata Updates]. The process logic for transitioning between states is as follows: [Users can mint public NFTs by providing the required payment until the supply depletes. The owner can mint private NFTs to specified addresses. The base URI can be updated by the owner at any time]. Main variables include [totalPublicSupply, remainingPublicSupply, and metadata mapping]. Main functions are [mintPublic, mintPrivate, and setBaseURI]. Key events include [BaseURIUpdated].

This smart contract is a **token sale contract**. The main functions include **purchasing tokens and withdrawing funds**. The contract operates in the following states: **Token Sale Active and Token Sale Ended**. The process logic for the transition of states involves: **users purchasing tokens during the active sale phase, which calculates the price based on USD value. After the sale period, the owner can withdraw the funds and tokens remaining in the contract**. The main variables include **cyceToken, WETH, usdPair, and tokenList**. The main functions are **buyToken, getUSDValue, and withdraw**. The main events include **purchase** which signals when a user buys tokens.
